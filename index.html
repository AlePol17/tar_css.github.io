CSS

Definición del concepto CSS

Sus siglas significa Cascade Style Sheets, basicamente es un lenguaje de estilos en cascada que otorga un diseño a las paginas web.

Regla de CSS Detallado

Para empezar en CSS debemos tener encuenta la siguiente regla que debe cumplir, pues se debe tener un selector , este debe ser declarado, asimismo se le debe asignar una propiedad al selector y por último debemos tener un valor para la propiedad.

Ejemplo:

h1: { --> Selector
color :red} --> Declaración
color: --> Propiedad
red: --> Valor
